Q1.Diffrentiate between flex-row and column ?

flex-row:The items start from the edge of the page it means it works on the main axis from left to right.
flex-column:The items are aligned from top to bottom.

Q2. Explain inline, internal and external stylesheet?

External CSS: External css is the type of css in which we create an external css file and link it with the html page.

Internal CSS: Internal CSS is the type of css in which style tag is placed inside the head tag.

Inline CSS: Inline css is the type of css in which style tag is placed inside the html tags.

Q3.Justify-content allows you to do what?

Jutsify-content define how the browser distributes space between and around content items along the main-axis of a flex container, and the minor axis of a grid container.

Q4.Differentiate between absolute and relative positioning?

The position property specifies the type of positioning method used for an element
  • Relative
  • Absolute
  • Static
  • fixed
  • Sticky
An element with position:relative; is positioned relative to its normal position whereas

Absolute positioning is positioned relative to the nearest positioned ancestor.

Q5.What is grid template column used for?

Grid template column specifies the number of columns and the size of columns of the grid.

Q6.What is z-index in css?

z-index is the property that specifies the stack order of an element. An element can have a positive or negative stack order.

Q7.What is the difference between padding and margin?

In CSS, a margin is the space around an element's border, while padding is the space between an element's border and the element's content.
